An iterative seed finding algorithm quantifies Voronoiness
N. Balkır Göka, Koray Yavuz, Emin Aliyev, Arash Mobaraki, Ali Javili, Seymur Jahangirov


This repository contains the code and data necessary to reproduce the figures for the paper "An iterative seed finding algorithm quantifies Voronoiness".

## Repository Structure

The repository is organized by figure, with each directory containing the code and data required to generate that specific figure.

- `Figure 1 Algorithm`: Visualizations and diagrams for the algorithm.
- `Figure 2 Convergence`: Code demonstrating the convergence properties of the algorithm under various conditions (weighted/unweighted, periodic/finite).
- `Figure 3 Run time`: Scripts for analyzing the runtime complexity and performance.
- `Figure 4 Degeneracy`: Examples and analysis of degenerate cases.
- `Figure 5 Global`: Global optimization and properties.
- `Figure 6 Decay`: Decay analysis.
- `Figure 7 Derivation`: Derivation-related visualizations.
- `Figure 8 Deviatorics`: Analysis of deviatoric components.
- `Figure 9 Wings`: Application to biological wing structures.
- `Supp Figure Convergence`: Supplemental data on convergence.
- `Supp Figure Panorpa`: Supplemental analysis for Panorpa (scorpionflies) wing patterns.
- `Supp Figure Spatial`: Supplemental spatial analysis.
- `Supp Figure Unweighted`: Supplemental unweighted analysis.

## Usage

Navigate into the respective figure folder. If there is a Jupyter Notebook (`.ipynb`), you can execute it to reproduce the corresponding figure and run the analyses.

## Requirements

To run the notebooks, you generally need:

- Python 3.x
- Jupyter Notebook or JupyterLab
- Standard scientific Python packages (e.g., NumPy, SciPy, Matplotlib)

_(Please install any additional dependencies as required by individual notebooks.)_
